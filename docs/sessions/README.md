# Session Notes

Chronological notes from Claude sessions. Add a new file for each significant session.

Format: `YYYY-MM-DD-topic.md`

Example: `2026-06-05-spiel-collection-atm-go.md`

## What to log

- What was done (phases completed, variables created/deleted/renamed)
- Any new gotchas or Figma API behaviors discovered
- Decisions made (naming choices, scope changes)
- Anything that should update a skill file
